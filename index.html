조금 더 빠른 HTML 작성을 위한 잡기술

개요

조금이라도 더 빠른 작성을 위한 잡기술. 이 페이지는 최대한 인터넷활용실습 첫 수업과 두번째 수업 수준의 진도를 바탕으로 제작.

보통 젠코딩 혹은 Emmet이라고 부르는 것과 자주 사용하는 단축키 몇 개 소개하려한다. 유래고 뭐고 그냥 사용법만 알려주려한다.

기초만 다루기 때문에 스스로 검색하며 찾아보고 직접 작성해보길 바란다.

    • 태그 작성법

    매우 간단하다. 태그 이름만 작성하고 엔터 누르면 된다.

    태그 안 내용 작성법

    '태그{내용}'식으로 작성하면 된다.

    태그가 들어있는 태그 작성법

    '태그>들어갈 태그' 식으로 작성하면 된다.

    속성이 있는 태그 작성법

    속성에 맞는 기호가 있다. 기초적인 기호만 설명할테니 다른 기호는 직접 찾아보길 바란다.

    (1) a 태그

    a 태그는 기본적으로 하이퍼링크를 걸어주는 태그이다. 때문에 기본적으로 'href='가 붙어서 나온다.

    그러나 a 태그는 하이퍼링크 종류별 작성법이 또 존재한다.

    이를테면 실습때 배운 url

    (2) id 부여

    태그에는 id라는것을 부여할 수 있다. 이는 추후에 배울 css와 js에서 활용할 예정이니 미리 작성법만 소개하려한다.

    '태그#id이름' 식으로 작성하면 된다.

    (3) class 부여

    태그에는 class라는것을 부여할 수 있다. 역시 이는 추후에 배울 css와 js에서 활용할 예정이니 미리 작성법만 소개하려한다.

    '태그.class이름' 식으로 작성하면 된다.

    여러개의 같은 모양 태그 작성법

    양식에 맞게 태그를 작성한뒤 소괄호로 묶어서 '*반복할 횟수'를 붙이고 엔터를 누르면 된다.

    여러 태그 덩어리가 있는 태그 작성법

    '태그+태그+태그+...'식으로 작성하면 된다.

    주의사항

    엔터 누르기 직전에 아래에 나오는 파란 박스안에 작성한 코드가 같지 않거나 커서가 다른곳이 눌리면 작성법이 작동하지 않는다.

    이럴때는 맨 마지막 글자를 지웠다가 다시 입력한 후 엔터를 누르면 된다.

    여는 중괄호 뒤에 *이 있어도 작성법이 작동하지 않는다.

    자주 사용하는 단축키

    (1) Home과 End

    줄의 맨 앞과 뒤로 이동할 수 있다.

    *ctrl과 함께 누르면 코드의 맨 처음 혹은 끝으로 이동할 수 있다.

    (2) Alt + 방향키

    해당 줄의 코드를 옮길 수 있다.

    *shift와 함께 누르면 해당 줄의 코드를 복사할 수 있다. (가장 좋아하는 단축키)

    (3) Alt + D

    원하는 단어 뒤에서 누르면 같은 단어들을 선택할 수 있다.

    *Alt + D를 누를때마다 하나씩 선택된다. 아무곳이나 클릭하면 원래대로 돌아온다.
